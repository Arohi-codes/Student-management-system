# 🎓 Student Management System

A menu-driven **Student Management System** built using **Python**. This project was created to practice Python fundamentals by implementing real-world CRUD (Create, Read, Update, Delete) operations.

---

## 🚀 Features

* ➕ Add Student
* 👀 View All Students
* 🔍 Search Student
* ✏️ Update Student Details
* ❌ Delete Student
* 🏆 Show Topper
* 📊 Calculate Class Average Marks
* 👨‍🎓 Display Total Number of Students
* 🚪 Exit Program

---


4. Follow the on-screen menu.
